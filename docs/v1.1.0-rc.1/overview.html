    • Player::__construct - This class is not intended for direct usage.
    • Player::consume - Enables/Disables the consume mode
    • Player::crossfade - Sets crossfade to $seconds seconds.
    • Player::mixramp_db - Sets the threshold at which songs will be overlapped.
    • Player::mixramp_delay - Additional time subtracted from the overlap calculated by mixrampdb.
    • Player::random - Specified if MPD should play the queue in random order
    • Player::repeat - Specifies if MPD should start from the top again when reaching the end of the queue.
    • Player::volume - Sets volume to $volume or returns the current volume if $volume is omitted.
    • Player::single - Enables/Disables the single-mode. If enabled MPD will play the same song over and over.
    • Player::replay_gain_mode - Specifies whether MPD shall adjust the volume of songs played using ReplayGain tags.
    • Player::replay_gain_status - Get the current replay gain. Currently only the variable replay_gain_mode is returned by MPD.
    • Player::current_song - Returns an associative array containing information about the currently playing song.
    • Player::next - Plays the next song in the Queue
    • Player::pause - Pause or resume playback.
    • Player::play - Plays the song position $pos in the Queue
    • Player::play_id - Begins playing the playlist at song $id
    • Player::previous - Plays the previous song in the Queue
    • Player::seek - Seeks to $seconds of song $songpos in the Queue.
    • Player::seek_id - Seeks to $seconds of song $songid. Same as Player::seek() but with a song ID instead of a position.
    • Player::seek_cur - Seeks to $seconds of the current song.
    • Player::stop - Stops playing.
    • Playlist::__construct - This class is not intended for direct usage.
    • Playlist::exists - Function to determine if the specified playlist exists.
    • Playlist::get_songs - Returns a list of all songs in the specified playlist.
    • Playlist::load - Loads the specified playlist into the Queue.
    • Playlist::add - Adds $uri to the specified playlist at position $pos.
    • Playlist::add_search - Search for songs using Filter and add them into the Playlist at position $pos.
    • Playlist::clear - Removes all songs from the specified playlist.
    • Playlist::remove_song - Deletes $songpos from the specified playlist.
    • Playlist::move_song - Moves the song at position $from in the specified playlist to the position $to.
    • Playlist::rename - Renames the specified playlist to $new_name
    • Playlist::delete - Removes the specified playlist from the playlist directory.
    • Playlist::save - Saves the queue to the specified playlist in the playlist directory
    • Queue::__construct - This class is not intended for direct usage.
    • Queue::add - Adds the file $uri to the queue (directories add recursively). $uri can also be a single file.
    • Queue::add_id - Adds a song to the playlist (non-recursive) and returns the song id.
    • Queue::add_search - Same as search() but adds the songs into the Queue at position $pos.
    • Queue::add_find - Same as find() but this adds the matching song to the Queue.
    • Queue::clear - Clears the queue
    • Queue::delete - Deletes a song or a range of songs from the queue
    • Queue::delete_id - Deletes the song with ID $songid from the Queue
    • Queue::move - Moves the song at $from to $to in the queue
    • Queue::move_id - Moves the song with $from (songid) to $to (playlist index) in the queue.
    • Queue::find - Same as Queue::search but case-sensitive
    • Queue::get_id - Returns an associative arrays containing information about the song with ID $songid.
    • Queue::get - Returns information about all or specific songs in the Queue.
    • Queue::search - Search the queue for matching songs.
    • Queue::changes - Returns an array of changed songs currently in the playlist since $version.
    • Queue::prio - Sets the priority of given songs to $priority.
    • Queue::prio_id - Sets the priority of Song ID $id to $priority.
    • Queue::range_id - Set's the portion of the song that should be played.
    • Queue::shuffle - Shuffle the Queue.
    • Queue::swap - Swap two songs in Queue. By Position.
    • Queue::swap_id - Swap two songs in Queue. By ID.
    • Queue::add_tag_id - Adds a tag to the specified song
    • Queue::clear_tag_id - Removes a tag from the specified song
