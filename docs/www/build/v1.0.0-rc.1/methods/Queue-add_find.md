```php
Queue::add_find(FloFaber\MphpD\Filter $filter, string $sort = '', array $window = Array, int $pos = -1) : bool
```

#### Parameters

*  \Filter $filter
*  string $sort
*  array $window
*  int $pos Optional. If specified the matched songs will be added to this position in the Queue.
