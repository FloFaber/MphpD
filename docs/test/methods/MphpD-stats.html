stats

Returns the value of the specified key from MPD's stats. 

MphpD::stats(array $items = []) : void

Parameters

$items

Returns void

array|false|int|null Returns `false` on error `string`, `int` or `null` if $items contains only one item. If it does not exist `null` will be returned instead. Otherwise, an associative array containing all available (or specified) stats.
