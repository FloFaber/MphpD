status

Returns the value of the specified key(s) from MPD's status. 

MphpD::status(array $items = []) : void

Parameters

$items

Returns void

array|false|int|float|null Returns `false` on error `string`, `int`, `float` or `null` if `$items` contains only one item. If it does not exist `null` will be returned instead. Otherwise, an associative array containing all available (or specified) keys.
